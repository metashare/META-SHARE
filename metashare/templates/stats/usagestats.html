This panel shows which metadata have been used to describe the META-SHARE linguistic resources.

Each metadata defined in the schema can be either "required", "recommended" or "optional".
The metadata usage statistics reports two counters. The first counter says how many times a metadata has been filled in with a certain value, while
the second counter says the number of resources for which the metadata has been used.
For instance
 28/12 
means that the corresponding required metadata has been filled in 28 times (possibly with redundant values), and that 12 different resources
have been described with that metadata.

You can click on a metadata name to see a table with all the actual values used to fill in that metadata.
